# C Retail Inventory Management System

A C programming academic group project developed as a retail store inventory management application. The system is designed to manage products, inventory, categories, suppliers, transactions, and file operations using modular programming, linked lists, and text-file handling.

## Project Overview

This project simulates a retail inventory management environment in C. It uses a menu-driven structure and modular source/header files to organize different parts of the system, including product records, stock handling, category and supplier management, user transactions, and file operations.

## Features

- Product management
- Inventory and stock management
- Category and supplier management
- User transaction handling
- File-based data operations
- Menu-driven console interface
- Linked list-based dynamic data handling

## Technologies Used

- C Programming
- File Handling
- Linked Lists
- Modular Programming
- Header and Source File Organisation

## Files Included

- `main.c` — main program flow
- `Inventory.c / Inventory.h` — inventory-related logic
- `Product.c / Product.h` — product-related logic
- `Category_Supplier.c / Category_Supplier.h` — category and supplier handling
- `FileOperations.c / FileOperations.h` — file-based operations
- `User_Transaction.c / User_Transaction.h` — user transaction handling
- `Common.c / Common.h` — shared utility logic
- `Files/` — project data files
